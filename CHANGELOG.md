# Changelog

Все значимые изменения в проекте будут документированы в этом файле.

## [2.1.0] - 2025-01-21
### Changed
- Упрощена админ-панель: удалены разделы Templates, Projects (админка), Advanced Search, Audit, Integrations
- Оптимизирована структура проекта для production

### Added
- Добавлена секция Projects в публичное портфолио на главной странице
- Создана страница детального просмотра проекта `/projects/[slug]`
- Улучшена навигация и структура портфолио

### Fixed
- Исправлена ошибка `templates.map is not a function`
- Исправлены предупреждения CSS linter для Tailwind
- Исправлено предупреждение NODE_ENV через добавление cross-env

## [2.0.0] - 2025-20-11
### Added
- Система тем (темная/светлая) с провайдером и переключателем
- Динамическая система проектов с БД и детальными страницами
- Версионирование заметок с историей изменений
- Экспорт заметок в Markdown и PDF
- Система шаблонов для заметок
- Совместный доступ к заметкам между пользователями
- Трекинг активности и аналитика использования
- Дашборд аналитики с графиками и статистикой
- Планировщик задач с приоритетами и дедлайнами
- Система напоминаний с уведомлениями
- Автоматические бэкапы БД и файлов
- Интеграция GitHub API для отображения активности
- Интеграция LeetCode для отображения статистики
- Импорт данных из Notion, Obsidian и других сервисов
- Система webhooks для внешних интеграций
- Публичное REST API с аутентификацией через API ключи
- Система горячих клавиш для основных действий
- Расширенный поиск с фильтрами и сортировкой
- Система избранного для быстрого доступа
- TechnologyBadge компонент для отображения технологий

### Changed
- Миграция с SQLite на PostgreSQL для production
- Обновлена структура базы данных с новыми моделями
- Улучшена производительность и оптимизация

### Infrastructure
- Docker конфигурация для production
- Docker Compose с PostgreSQL
- Nginx конфигурация с SSL
- GitHub Actions для CI/CD
- Скрипты для автоматического развертывания
- Полная документация по развертыванию

### Security
- Security headers в Next.js и Nginx
- Улучшенная обработка ошибок
- Rate limiting для API
- Безопасная работа с SSL сертификатами

## [1.0.0] - 2025-10-10

### Initial Release
- Базовый функционал заметок, файлов, vault и календаря
- Аутентификация через NextAuth
- 2FA поддержка
- Аудит безопасности
