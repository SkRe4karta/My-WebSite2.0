# Project Plan\n\n## Goals\n- Public portfolio (Home, Projects, About) with dark/green theme, parallax and scroll animations.\n- Private admin area at /admin with notes, files, vault, idea journal, settings.\n- Single-user auth with password + TOTP (NextAuth credentials).\n- SQLite via Prisma for content, file metadata, vault ledger, journal events.\n- Local storage folders: storage/uploads, storage/vault (encrypted).\n- API routes for notes, files, vault, journal, settings.\n- Dockerized deployment + nginx reverse proxy + certbot.\n\n## Architecture\n- Next.js App Router + Tailwind for UI; components under src/components.\n- NextAuth for sessions; middleware protects /admin.\n- Prisma ORM; migrations stored in prisma/migrations.\n- File handling via app/api/files route using FormData uploads.\n- Vault encryption helper with AES-256-GCM, keys from env.\n- Idea journal calendar computed client-side; server stores events.\n- Logging & rate limiting middleware wrappers for APIs.\n\n## Roadmap\n1. Scaffold public pages with hero, projects, about content.\n2. Create shared UI kit (Sidebar, cards, buttons, parallax sections).\n3. Configure NextAuth, Prisma, database schema, seed script.\n4. Build admin pages: dashboard, notes (markdown), files (drag-drop), vault, journal, settings.\n5. Implement API routes for CRUD, file storage, encryption, search.\n6. Add Docker/Docker-compose, nginx, certbot automation, deployment guide.\n7. Polish: animations, themes, env samples, backup scripts.\n
